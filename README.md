# GPT-to-web

Creating a Chatbot using the Assistants API of OpenAI
Steps:
1. Create a assistant.json file which will contain the assistant_id
2. Use your files as docs.pdf or if you want to use multiple files change the code as per your requirement.
3. Create a .env file containing the OPENAI_API_KEY
4. Install all the requirements
5. Use the command `python main.py`
